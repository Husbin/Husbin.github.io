webpack整体配置介绍

发表于 2018-07-04 | 阅读次数:

主要配置参数

entry

项目的入口和起点，指示webpack应该使用哪个模块，来作为构建项目依赖图的开始。

output

output告诉webpack在哪里输出打包编译好的文件，并且规定文件的命名。

module

在module可以配置loader的规则，来规定如何对源代码进行转换。

css-loader，url-loader，babel-loader

resolve

设置模块如何被解析。

plugins

plugin 用于扩展webpack的功能。它直接作用于 webpack，扩展了它的功能。

解决一些loader无法解决的问题。
